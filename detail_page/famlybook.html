Famly Book

Deskripsi Project

Famly Book adalah sistem pencatatan laundry berbasis yang membantu pengelolaan transaksi, pencatatan pelanggan secara efisien dan terstruktur.

Fitur Utama

  • Input transaksi laundry
  • Manajemen data pelanggan
  • Laporan transaksi

Teknologi

Java MySQL
